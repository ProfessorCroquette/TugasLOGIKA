TECHNICAL IMPLEMENTATION
================================================================================

DATA INTEGRATION:
  Source: base.csv (91,221 lines)
  Contains: Provinces, Kabupaten/Kota, Kecamatan
  Coverage: 34 provinces, 400+ kabupaten/kota, 7,000+ kecamatan
  Loading: One-time at startup (~50-100ms)
  Caching: In-memory for fast lookups
  Fallback: Default province list if CSV unavailable

MODULE STRUCTURE:

  IndonesianRegions (Singleton):
    ├── _load_data()
    ├── _load_from_csv()
    ├── _load_default_provinces()
    ├── get_province(code)
    ├── get_kabupaten(code)
    ├── get_kecamatan(code)
    ├── get_all_provinces()
    ├── get_all_kabupatens()
    ├── get_kabupatens_by_province()
    └── get_kecamatan_by_kabupaten()

  NIKParser (Static Methods):
    ├── parse_nik(nik) → Dict
    ├── get_readable_nik_info(nik) → str
    └── get_summary_nik_info(nik) → Dict

RETURNED DATA (16-digit NIK):
  {
    'nik': str,                    # Original NIK
    'nik_formatted': str,          # Formatted: AA BB CC DD MM YY ZZZZ
    'province_code': str,          # 2 digits
    'province_name': str,          # Full name from CSV
    'kabupaten_code': str,         # 2 digits
    'kabupaten_name': str,         # Full name from CSV
    'kecamatan_code': str,         # 2 digits
    'kecamatan_name': str,         # Full name from CSV
    'birth_day': int,              # 1-31
    'birth_month': int,            # 1-12
    'birth_month_name': str,       # Januari-Desember
    'birth_year': int,             # 1900-2028
    'birth_date': str,             # "17 Agustus 1990"
    'gender': str,                 # "Laki-laki" or "Perempuan"
    'is_female': bool,             # Gender flag
    'sequential_number': str,      # "0001"-"9999"
    'age': int                     # Calculated age
  }
