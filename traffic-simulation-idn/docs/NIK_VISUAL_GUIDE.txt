================================================================================
INFORMATION EXTRACTION
================================================================================

From Single NIK: 3275075708900004

The system automatically extracts and displays:

1. GENDER
   Input:     57 (from positions 7-8)
   Detection: 57 > 40 ? YES
   Output:    "Perempuan (Female)"

2. BIRTHDAY
   Input:     570890 (from positions 7-12)
   Parsing:   57 (day code) | 08 (month) | 90 (year)
   Extraction:
     Day:     57 - 40 = 17
     Month:   08 = Agustus
     Year:    90 = 1990
   Output:    "17 Agustus 1990"

3. PROVINCE
   Input:     32 (from positions 1-2)
   Lookup:    32 = Maluku Utara
   Output:    "32 - Maluku Utara"

4. AGE
   Input:     Birth year = 1990, Current year = 2026
   Calculation: 2026 - 1990 = 36
   Output:    "36 tahun"

5. REGISTRATION
   Input:     0004 (from positions 13-16)
   Output:    "Pendaftar ke: 0004"

All automatically calculated from 16 digits!

Note: Day is 17 (without +40 offset because he's male)

================================================================================
COMPARISON: FEMALE vs MALE NIK
================================================================================

FEMALE EXAMPLE:
  NIK:        3275075708900004
  Day Digits: 57 (indicates female)
  Actual Day: 57 - 40 = 17
  Gender:     Perempuan (Female)
  Birth Date: 17 Agustus 1990

MALE EXAMPLE:
  NIK:        3210031715031989
  Day Digits: 17 (indicates male, no offset)
  Actual Day: 17 (no subtraction)
  Gender:     Laki-laki (Male)
  Birth Date: 17 Maret 1989

VISUAL:
  Female day 17 → encoded as 57 in NIK
  Male day 17   → encoded as 17 in NIK
                  Difference: 40 offset for females

================================================================================

1. Main Table
   └─ User sees violation with owner name
   └─ User clicks "Lihat Detail"

   └─ Shows vehicle, owner, registration, violation, fine

      ├─ Owner Name: [name]

   └─ User clicks button

   └─ Shows all NIK information

   └─ User clicks OK

   └─ User can view other sections
