DECODING EXAMPLES
================================================================================

EXAMPLE 1: 3275075708900004
  32 = JAWA BARAT (Province)
  75 = KOTA BEKASI (Kabupaten/Kota)
  07 = Bantargebang (Kecamatan)
  57 = Day 17, Female (57-40=17)
  08 = Agustus (Month)
  90 = 1990 (Year)
  0004 = Registration 4

  Result:
  - Province: JAWA BARAT
  - City: KOTA BEKASI
  - District: Bantargebang
  - Birth Date: 17 August 1990
  - Gender: Female (Perempuan)
  - Age: 36 years (2026-1990)
  - Registration: 4th person registered

EXAMPLE 2: 3375061503950001
  33 = JAWA TENGAH (Province)
  75 = KOTA SEMARANG (Kabupaten/Kota)
  06 = Gajahmungkur (Kecamatan)
  15 = Day 15, Male (≤31, no offset)
  03 = Maret (Month)
  95 = 1995 (Year)
  0001 = Registration 1

  Result:
  - Province: JAWA TENGAH
  - City: KOTA SEMARANG
  - District: Gajahmungkur
  - Birth Date: 15 March 1995
  - Gender: Male (Laki-laki)
  - Age: 31 years (2026-1995)
  - Registration: 1st person registered

================================================================================
GENDER DETECTION
================================================================================

FEMALE INDICATOR:
  Day code > 40 (i.e., 41-71)
  Examples: 41, 42, ..., 57, ..., 71
  To get actual birth day: Subtract 40 from code
  Example: 57 = Female born on day 17 (57-40=17)

MALE INDICATOR:
  Day code ≤ 40 (i.e., 01-31, 32-40 unused)
  Examples: 01, 02, ..., 15, ..., 31
  To get actual birth day: Use code as-is
  Example: 15 = Male born on day 15

VALIDATION:
  Valid female days: 41-71 (represents 1-31)
  Valid male days: 01-31
  Invalid: 32-40 (unused range)
  Invalid: 72+ (exceeds female range)

================================================================================
YEAR CONVERSION
================================================================================

RULES:
  YY 00-28 = 20YY (2000-2028)
  YY 29-99 = 19YY (1929-1999)

EXAMPLES:
  00 = 2000
  10 = 2010
  28 = 2028
  29 = 1929
  50 = 1950
  95 = 1995
  99 = 1999

RATIONALE:
  Threshold at 28/29 ensures realistic birth years
  No one born before 1929
  No one older than 97 years (2026-1929)
  Supports people from 1929 to 2028 (99+ years range)

================================================================================
QUICK REFERENCE FORMAT
================================================================================

NIK FORMAT: AA BB CC DD MM YY ZZZZ

  AA (2 digits) = Province Code (11-94)
  BB (2 digits) = Kabupaten/Kota Code (01-99)
  CC (2 digits) = Kecamatan Code (01-99)
  DD (2 digits) = Birth Day (01-31 male, 41-71 female)
  MM (2 digits) = Birth Month (01-12)
  YY (2 digits) = Birth Year last 2 digits (00-99)
  ZZZZ (4 digits) = Sequential Registration (0001-9999)

TOTAL: 16 digits

================================================================================
COMMON MISTAKES TO AVOID
================================================================================

❌ Day > 31 for males (invalid)
✓ Day 01-31 for males

❌ Day < 40 for females (invalid, looks male)
✓ Day 41-71 for females

❌ Month > 12 (invalid)
✓ Month 01-12

❌ Province code 00, 10, 20 (invalid)
✓ Valid codes: 11-19, 21, 31-36, 51-53, 61-65, 71-76, 81-82, 91, 94

❌ Using 99 for December (invalid)
✓ Use 12 for December

❌ Forgetting gender offset for females
✓ Add 40 to day for female registration
