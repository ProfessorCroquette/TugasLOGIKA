1. **Navigate to project**

## Customization

### Change Speed Limit
Edit `config.py`:
```python
SPEED_LIMIT = 80  # Changed from 75
```

### Change Fine Amounts
Edit `config.py`:
```python
FINES = {
    "LEVEL_1": {"min": 76, "max": 90, "fine": 150},  # Changed from 100
    ...
}
```

### Change Vehicle Generation Frequency
Edit `config.py`:
```python
SIMULATION_INTERVAL = 5  # Generate every 5 seconds instead of 10
```

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Dashboard clears too fast | The display updates every 5 seconds - this is normal |
| Can't press 'q' to quit | Try `Ctrl+C` instead |
| No data files created | Check you have write permissions and space on disk |
| Error with imports | Make sure you're in the project root directory |
