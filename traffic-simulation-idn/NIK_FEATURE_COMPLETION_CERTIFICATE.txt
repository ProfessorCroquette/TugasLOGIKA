FEATURE OVERVIEW
================================================================================

WHAT WAS IMPLEMENTED:
  A button in the GUI that allows users to view detailed information
  about vehicle owners extracted from their 16-digit NIK (National ID).

HOW IT WORKS:
  1. User views violation in GUI table
  2. Clicks "Lihat Detail" to see violation details
  3. In owner information section, clicks "Lihat Detail NIK" button
  4. Dialog displays complete decoded NIK information:
     • Formatted NIK with spacing
     • Province code and province name
     • District and subdistrict codes
     • Birth date (day, month, year)
     • Gender (Male/Female)
     • Current age
     • Registration order number
     • Explanation of format

INFORMATION EXTRACTED:
  From a 16-digit NIK, the system automatically extracts:
  • Gender: Detected from day field (male ≤31, female >40)
  • Birthday: Extracted from day, month, year fields
  • Province: First 2 digits mapped to province name
  • Age: Calculated from birth year
  • Registration: Sequential number (0001-9999)

KEY FEATURES:
  ✓ Automatic gender detection
  ✓ Birthday extraction with month names in Indonesian
  ✓ All 34 Indonesian provinces mapped
  ✓ Age calculation
  ✓ Readable, professional dialog display
  ✓ Comprehensive error handling
  ✓ No external dependencies
