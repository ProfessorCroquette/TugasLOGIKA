USAGE SUMMARY
================================================================================

To use in your application:

1. GENERATE VEHICLES:
   from utils.generators import DataGenerator
   vehicles = DataGenerator.generate_vehicle_batch()

2. ACCESS PLATE INFO:
   vehicle.license_plate       # "B 1234 ABC"
   vehicle.plate_color         # "BLACK"
   vehicle.plate_type          # "PRIBADI"
   vehicle.vehicle_category    # "Pribadi"

3. FORMAT FOR DISPLAY:
   from utils.gui_vehicle_formatter import VehicleDisplayFormatter
   info = VehicleDisplayFormatter.get_vehicle_info(vehicle)
   summary = VehicleDisplayFormatter.get_vehicle_summary(vehicle)

4. APPLY COLOR STYLING:
   from utils.gui_vehicle_formatter import PLATE_COLOR_MAP
   colors = PLATE_COLOR_MAP[vehicle.plate_color]

5. FILTER VEHICLES:
   private = VehicleDisplayFormatter.filter_vehicles_by_category(
       vehicles, 'Pribadi'
   )

6. GET STATISTICS:
   stats = VehicleDisplayFormatter.get_statistics(vehicles)
